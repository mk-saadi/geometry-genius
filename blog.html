1. What is a DOM?

Document Object Model It is a programming interface for web documents that allows developers to dynamically manipulate the contents and structure of web pages.


The DOM is an essential part of modern web development and is used in a variety of applications, including dynamic web pages, web applications, and mobile apps. It allows developers to create interactive and responsive user interfaces that can adapt to user input and data changes.

2. How to select element from DOM?

In order to select an element from DOM, you can use the DOM API methods provided by JavaScript. Here are a few common methods to select an element from the DOM:

  1. getElementById
  2. getElementsByClassName
  3. getElementsByTagName
  4. querySelector
  5. querySelectorAll

3. What is Event Bubble?

Event Bubble is a concept in the DOM. It happens when an element receives an event, and that event bubbles up to its parent and ancestor elements in the DOM tree until it gets to the root element

4. What is Callback Function?

A callback function is a function passed into another function as an argument, which is then invoked inside the outer function to complete some kind of routine or action.
